# PRAWDA - Offline translate english-russian

## Compiling and Installing

     Compilation and installation is as usual:

```bash
    $ tar -xazf prawda-0.3.0.tar.xz
    $ cd prawda-0.3.0
    $ cd bin
    $ make
    $ sudo make install
```

## Program description
     
    Offline translate english-russian
    
## Dependencies

    This program requires the xterm.

    Run the executable:

```bash
    $ koi8rxterm prawda
```
